# Passimx Server instance

> An instruction how to start server instance easy and quick .

## Getting Started

```bash

# Clone the repository
git clone https://github.com/Passimx/server-instance.git

# Enter the directory
cd server-instance

# Give permission to use your machine disk
sudo chown -R 1001:1001 ./data

# Give permission to run .sh files (for start.sh)
chmod -R 777 ./

# Run start.sh script
./start.sh
```
